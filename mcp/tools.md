# Jarvis MCP Tools

## robot_status()

Returns the current status of the robot.

## wikipedia_search(topic)

Retrieves factual information about a topic.

## web_search(query)

Performs real-time web searches.

## latest_ai_news()

Retrieves the latest AI news.

## robotics_news()

Retrieves the latest robotics news.

## weather(city)

Retrieves current weather information.

## remember(key, value)

Stores information in memory.

## recall(key)

Retrieves stored information.

## daily_briefing()

Provides a combined weather, AI news, and robotics news summary.

## project_status()

Returns current project information.

## save_project_idea()

Stores project ideas.

## get_project_ideas()

Retrieves saved project ideas.

## troubleshooting_help()

Provides troubleshooting assistance.
