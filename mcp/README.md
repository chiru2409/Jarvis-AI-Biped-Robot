# Jarvis MCP Server

The MCP server extends Jarvis with custom tools that provide:

- Web Search
- AI News
- Robotics News
- Weather Information
- Memory Storage
- Project Assistance
- Troubleshooting Support

## Installation

```bash
pip install -r requirements.txt
```

## Environment Variables

Create a `.env` file:

```env
TAVILY_API_KEY=your_api_key_here
```

## Run

```bash
python jarvis_tools.py
```
