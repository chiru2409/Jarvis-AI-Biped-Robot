# Project Story

Jarvis started as an idea to build an affordable AI-powered robotic companion using commonly available hardware.

The goal was to combine:

- Robotics
- Artificial Intelligence
- Voice Interaction
- Memory
- Internet Access

into a single ESP32-S3 powered biped robot.

The robot is capable of:

- Voice conversations
- Long-term memory
- Web search
- AI news retrieval
- Robotics news retrieval
- Weather information
- Project assistance

Jarvis continues to evolve with future plans including vision, autonomous navigation, smart device control, and advanced robotic behaviors.
